# deps-zed

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](../../LICENSE)

Zed editor extension for [deps-lsp](https://github.com/bug-ops/deps-lsp).

> [!NOTE]
> This crate is compiled to WASM and distributed via Zed Extensions marketplace.

## Installation

Install the **Deps** extension from Zed Extensions marketplace.

## Development

```bash
# Build WASM extension
cargo build --target wasm32-wasip1
```
